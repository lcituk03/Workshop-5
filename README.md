
# Workshop 5 – Time Tracker

## Opis projektu

Aplikacja **Time Tracker** służy do zarządzania zadaniami oraz mierzenia czasu potrzebnego na wykonanie poszczególnych operacji w ramach zadania.

Użytkownik może:
- dodawać zadania (tytuł + opis),
- dodawać operacje do zadań,
- zapisywać czas spędzony na każdej operacji,
- zamykać zadania,
- usuwać zadania i operacje.

Dane aplikacji nie są przechowywane lokalnie — aplikacja komunikuje się z **zewnętrznym REST API**.
